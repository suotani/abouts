プライバシーポリシー

対象サービス：宅建デイリー / 最終更新日：2026-01-30

本プライバシーポリシーは、「宅建デイリー」（以下「本アプリ」）におけるユーザー情報の取扱いについて定めるものです。

1. 取得する情報

  • 通知配信用トークン：Firebase Cloud Messaging（FCM）により発行される端末トークン等
  • 利用状況に関する情報：アプリの利用状況（イベント等）を計測するための情報（Firebase Analytics を利用する場合）
  • 端末・アプリ情報：OS種別・バージョン、アプリバージョン等
  • クラッシュ情報：不具合解析のためのクラッシュログ等（Firebase Crashlytics を利用する場合）

※ 本アプリが提供する機能により、ユーザーが入力した情報（例：メモや設定等）を取得・保存する場合があります。該当する場合は本アプリ内の表示または更新版ポリシーにて明示します。

2. 利用目的

  • 本アプリの機能提供（通知配信を含む）
  • 品質改善（不具合修正、利用傾向の把握、機能改善）
  • 問い合わせ対応
  • 不正利用の検知・防止

3. 第三者提供・外部送信

本アプリは、機能提供や品質改善のため、Google LLC が提供する Firebase 等の外部サービスを利用することがあります。 これら外部サービスにより、上記「取得する情報」が外部送信・処理される場合があります。

利用する主な外部サービス（例）

  • Firebase Cloud Messaging（通知配信）
  • Firebase Analytics（利用状況の計測）
  • Firebase Crashlytics（クラッシュ解析）
  • Cloud Firestore / Cloud Functions（データ保存・サーバー処理）

Firebase のデータ取扱いについては Google のポリシー等をご確認ください。

4. 収集情報の管理

取得した情報は、適切な安全管理措置を講じて管理します。法令に基づく場合を除き、本人の同意なく第三者へ開示・提供することはありません。

5. 保存期間

取得した情報は、利用目的に必要な範囲および期間に限り保存し、不要となった場合は適切な方法で削除または匿名化します。

6. ユーザーによる設定・停止

  • 通知：端末の設定から通知をオフにできます。
  • 分析：本アプリに分析の無効化設定がある場合は当該設定から停止できます（実装していない場合もあります）。

7. データ削除の درخواست

ユーザー情報の削除をご希望の場合は、下記「お問い合わせ先」までご連絡ください。本人確認のため、合理的な範囲で追加情報の提供をお願いする場合があります。
